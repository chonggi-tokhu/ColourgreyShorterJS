2-1. methods and properties of objects of variables that their data types are array I add

(an array).returnAllInOneArr()

this function returns one array that has no properties that are array. if a child property in parent object array is array,child of child properties in the child array property are in parent object array. my english is terrible so read code below to understand my words. 

                                                
var myarray = [1,['a',[2,3,4],'b',5,'c','d'],['e'],6,7,8,[9,'f','g'],'h',0];
var mynewarray = myarray.returnAllInOneArr();
/* mynewarray is [1,'a',2,3,4,'b',5,'c','d','e',6,7,8,9,'f','g','h',0] here. */

It takes no parameters. It does not change original array.

2-2. methods and properties of objects of variables that their data types are string I add

(a string variable).splitByIdx()

string.split method splits at each index of character split method takes as a parameter. but splitByIdx method splits at index splitByIdx method takes as a parameter that is an integer.

                                                
var mystring ="Hello World!";
var mynewstring = mystring.splitByIdx(3);
/* mynewstring is ["hel", "lo World!"] */

It does not change original string.



(a string variable).insertOn()

this method inserts string from first parameter at integer from second parameter to parent string object.

                                                
var mystring = "Hello World!";
var mynewstring = mystring.insertOn(" ",4).insertOn("f",6);
/* mynewstring is "Hell of World!" */

It does not change original string.
